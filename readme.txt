Integration between WordPress and Leads Rocks! CRM & Lead Management application.

== Description ==

An integration between your WordPress contact forms and [Leads Rocks!](https://leads.rocks) application.
Gather all incoming leads from the most popular contact forms plugins and send them to [Leads Rocks!](https://leads.rocks).

The plugin enables you to quickly and smartly map your contact forms fields to your campaigns on [Leads Rocks!](https://leads.rocks).

Currently the plugins communicates with the most popular contact forms plugins:

* Contact Form 7
* Gravity Forms
* Ninja Forms
* Formidable
* Pojo Forms
* And more soon...

> You can contribute by simply [forking me on GitHub](https://github.com/dorzki/Leads-Rocks-WordPress).

== Installation ==

Install the plugin by:

1. Upload `leads-rocks` directory to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Go to `Leads Rocks!` page and setup the integration.

== Frequently Asked Questions ==

= What is Leads Rocks? =

[Leads Rocks!](https://leads.rocks) is a Lead Management & CRM application for Small and Medium businesses.

= Is Leads Rocks! free? =

No, [Leads Rocks!](https://leads.rocks) is an application on monthly subscription base, you may see [Pricing](https://leads.rocks/#pricing) page for more information.

= Does the plugin collect UTM parameters? =

Yes it does!

= What contact form plugins does the plugin integrate with? =

Currently the plugin enables you to integrate with the following plugins:

* Contact Form 7
* Gravity Forms
* Ninja Forms
* Formidable
* Pojo Forms
